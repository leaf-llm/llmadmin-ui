# LLM Admin — 省钱可靠模型搭档

统一调用 80+ 大模型，桌面端本地管理，省钱又省心。

## 特性

- 🖥️ **桌面端应用** — 基于 Neutralino.js 的跨平台桌面客户端（默认窗口 1280×840），开箱即用
- 🔌 **统一接口** — HTTP 统一调用 80+ 大模型，支持**图像生成**和**文本生成**
- 🔄 **协议转换** — 同时兼容 OpenAI `/v1/chat/completions` 和 Anthropic `/v1/messages` 协议，自动互转
- 🧭 **自定义路由** — 声明式模型路由、**故障转移**和**负载均衡**策略
- 🤖 **智能路由** — 按文本 / 图像 / 视频 / 音频 模态智能分配模型
- 📊 **用量统计** — 内置管理界面，实时查看模型用量和消费统计
- 🔌 **插件系统** — 支持客户端缓存、Guardrails 等扩展能力
- 🔒 **本地部署** — 所有数据保留在本地，无需外部 SaaS

## 快速开始

### 下载桌面端

从 [Releases](https://github.com/your-org/llm-admin/releases) 下载对应平台的桌面端应用，开箱即用。

### 或从源码运行

```bash
# 1. 克隆仓库
git clone <repo-url>
cd llm-admin

# 2. 安装依赖
npm install
cd src-gateway && npm install && cd ..

# 3. 启动开发模式
npm run dev:ui        # 启动 Web UI（端口 5173）
npm run bun:dev       # 启动网关（端口 8700）

# 或构建桌面端
npm run build:desktop
```

> 本桌面端是 llmadmin-core 的前端界面，用于可视化修改 `conf.json` 配置、管理提供商和路由策略等。

## 开发

```bash
npm run dev:ui              # 启动 UI 开发服务器（端口 5173）
npm run build:gateway       # 构建网关
npm run build:desktop       # 构建桌面端应用
npm run build:all           # 构建全部
npm run dev:desktop         # 桌面端开发模式
```
